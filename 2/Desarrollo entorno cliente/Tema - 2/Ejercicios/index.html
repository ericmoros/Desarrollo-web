Hoja 1

Considera para los siguientes ejercicios que todavía no sabemos acceder al DOM (Document Object Model) mediante JavaScript: • Para recoger datos de usuario, no podremos utilizar formularios, habrá que utilizar la función prompt. • Para mostrar datos por pantalla, tendremos que utilizar el método write del objeto document, que recibe como parámetro el html a insertar en la página). Ej: document.write (“Hola Mundo”);
